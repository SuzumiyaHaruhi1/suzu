Когда printf вызывается с пользовательским вводом без правильных спецификаторов формата, он интерпретирует этот ввод как строку формата. Это можно использовать для чтения содержимого памяти или для сбоя программы.
message - это пользовательский ввод, и если этот ввод включает спецификаторы формата, такие как %x, %s, printf будет обрабатывать эти спецификаторы, используя значения из стека.
Чтобы прочитать флаг, нужно найти, где переменная flag находится в стеке:

  1. Сначала вводим несколько спецификаторов %x, чтобы увидеть содержимое стека

    Заранее заготавливаем флаг и смотрим его hex-интерпритацию

    ```
%x %x %x %x %x %x %x %x %x %x
```
  2. Пробуем до тех пор пока не увидим флаг
  3. Чтобы прочитать строку из стека, нужно использовать спецификатор %s, который указывает printf воспринимать значение как указатель на строку. При этом нужно указать правильное смещение.
  4. Пробуем несколько вариантов до тех пор, пока не увидим флаг.
